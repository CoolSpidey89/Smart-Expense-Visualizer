# 💸 Smart Expense Visualizer

A modern **financial analytics dashboard** built with React.js that transforms raw CSV expense data into interactive insights, forecasts, and visual reports.

---

## 🚀 Features

- 📂 Upload expense data via CSV  
- 📊 Interactive Pie & Line charts for category and monthly trends  
- 🔮 Future spending prediction using time-series trend analysis  
- ⚠️ Anomaly detection for unusual spending spikes  
- 🎯 Dynamic filters (by category & month)  
- 💰 Budget planner with real-time over/under tracking  
- 🌗 Light/Dark theme toggle  
- 💾 Persistent storage using Local Storage  
- 📤 Export dashboard as PDF report  
- 🧠 Automated insights generation  

---

## 🧠 Tech Stack

- **React.js**
- **Chart.js** for data visualization
- **Tailwind CSS** for UI design
- **PapaParse** for CSV parsing
- **jsPDF + html2canvas** for PDF export

---

## 📈 Analytics Capabilities

This project includes:

- **Time-series forecasting** for next-period spending
- **Statistical anomaly detection** for irregular financial behavior
- **Category trend analysis**
